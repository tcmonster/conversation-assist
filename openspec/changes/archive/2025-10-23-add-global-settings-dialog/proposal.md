## Why
- 现有 UI 侧边栏提供“全局设置”入口但尚未实现实际功能,用户无法配置模型调用、上下文引用等核心能力。
- 产品要求本地化存储模型密钥与提示词上下文,需要统一设置面板管理并保留在浏览器 localStorage 中。

## What Changes
- 新增基于 shadcn Dialog 的全局设置弹窗,采用左侧菜单、右侧表单的结构,支持模型 API、参考信息、引用文本三个分区。
- 引入设置状态容器与 localStorage 持久化逻辑,确保刷新后仍能保留配置。
- 提供参考信息/引用文本的表格编辑能力,允许新增、编辑、删除条目并同步保存。

## Impact
- UI: 新增设置弹窗组件,调整侧边栏按钮交互以触发设置弹窗。
- State: 新建 Settings store/hook,负责读取/写入 localStorage。
- Tech Debt: 为后续导出导入奠定数据结构,需在后续任务中与 Export Manager 对接。
